# Star catalog attribution

`stars-bright.json` is derived from the [HYG Database](https://github.com/astronexus/HYG-Database)
maintained by David Nash. The HYG Database is released under the
[Creative Commons Attribution-ShareAlike 4.0](https://creativecommons.org/licenses/by-sa/4.0/)
license.

Only stars with apparent magnitude ≤ 6.0 are retained, formatted as
`[raHours, decDeg, mag, colorIndex|null]` tuples and sorted by brightness.

If you redistribute this file you must credit HYG and share derivatives
under the same CC BY-SA 4.0 terms.
